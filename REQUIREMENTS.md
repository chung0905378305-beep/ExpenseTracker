# 记账本 iOS App — 需求文档（v3.4 终稿）

> 经 6 轮第一性原理拆解与 28+ 项确认冻结。本版为开工基线；第 8、9 节为微观交互与技术默认，可微调。

## 1. 项目概述
原生 iOS 记账 + 资产管理 + 订阅 + AI 分析 + 负债债权工具。覆盖：多账户记账（期初余额、可配置转账、归档隐藏）、自定义大类/子类分类与标签、月度预算（带出上月）、循环/订阅自动生成、自动记账（规则/CSV导入/别家迁移/OCR/邮箱）、资产管理（股票·加密·基金·黄金·房产·现金外币，多源行情）、负债债权（信用卡透支/贷款/借入借出）、AI 分析（用你自己的 Key）、全局搜索、隐藏金额、CSV + 完整 JSON 备份、中+英 i18n；数据经 iCloud 多设备同步。目标 **App Store 上架**。

## 2. 平台与技术栈
- iOS 17+（iPhone 为主，iPad 自适应）；Swift + SwiftUI + SwiftData + CloudKit(iCloud 同步)
- 网络：URLSession（Yahoo 行情 / 天天基金·东方财富 / CoinGecko / 免费汇率 API / AI / 邮箱 IMAP·OAuth）
- 视觉：跟随系统浅色/暗色，蓝色强调色；分类图标用 SF Symbols
- App 锁：Face ID / Touch ID（设置可配触发时机，失败转密码兜底）
- 币种：可设**基础币种**（默认 CNY）；交易统一按基础币种记账；资产市值按基础币种 × 汇率折算展示
- 多语言：简体中文 + 英文 i18n

## 3. 功能清单
1. **首屏 = 记账页面**：App 打开直接是记账/快速录入页（第一 Tab）
2. **记账**：手动新增/编辑/删除（金额、类型、分类(子类)、账户、日期、备注、标签[]、附件[]）；记完回账单列表；默认记忆上次账户/分类；**支持关联退款**（一笔收入关联原支出并标注）
3. **转账**：账户间互转；资产变化可切换（默认计入两账户资产，净资不变），可关“计入资产”仅作备忘
4. **期初余额**：每个账户建账时填期初余额，作为净值与统计基准
5. **分类**：**大类/子类两级**（预设成树，如 餐饮→外卖/聚餐/咖啡）；内置 + 自定义 增/改/删 + 每分类可配「自动归类关键词」；最近/常用智能置顶
6. **标签**：跨分类多对多标签（如「出差」「宝宝」），可横向筛选
7. **账户**：多账户，分**资产类 / 负债类**；负债类含信用卡（账单日+还款日）、贷款、借入；借出作为资产类「应收」；**可归档隐藏（保留历史与余额）**
8. **预算**：月度总预算 + 分类预算；**新月份自动带出上月可微调**；超支发本地通知；总预算与分类预算**各自独立检测提示**
9. **循环记账（RecurringRule）**：按 日/周/月/年 自动生成；**无限生成，靠手动取消**；生成实例与模板独立，改实例不影响下次
10. **快速记账**：常用组合存模板，一键记
11. **列表**：按月分组 + 月度收支汇总；滑动删除、点击编辑、全局搜索（备注/商户/金额/分类/标签）；支持下钻（点月/分类看对应明细）
12. **统计**：月度收支/结余、储蓄率、环比、分类占比条形图、近 6 月趋势折线、年度 12 月汇总、净资产历史曲线；**支持自定义起止日区间**；图表可点进明细
13. **导出/备份**：CSV 导出 + 完整 JSON 备份（含图片）导出导入；支持从别家记账 App（钱迹/随手记等）经增强 CSV 列映射 + 模板预设导入
14. **自动记账**：
    - 规则自动归类（备注/商户关键词→分类，支持子类）
    - 循环自动生成（见 9）
    - 导入 CSV 账单（导入时映射列）+ 别家 App 迁移
    - 截图/PDF：本地 OCR（Vision，中+英）+ 你的 AI 整理归类
    - 邮箱授权拉取：IMAP（授权码）+ Gmail/Outlook OAuth；按发件人白名单/关键词筛账单 → AI 抽取（手动触发、前台）；**支持绑定多个邮箱**
    - 重复防护：自动生成按「来源+商户+金额+日期」hash 去重
    - 自动生成统一进「待确认」队列，编辑/批量确认/删除
15. **订阅管理（三种模式）**：周期自动扣款 / 一次性扣款 / 手动提醒（仅续费日前发本地通知）；列表展示即将续费；可标记取消（取消后停止生成/提醒）
16. **资产管理（通用资产表）**：类型 股票/加密/基金/黄金/房产/现金外币；字段 代码/名称/数量/成本价(手动)/当前价/价格时间/(可选)关联账户；搜索代码自动带出 + 手动兜底；**行情多源**：股票/ETF/美股/港股/A股→Yahoo（按市场拼 symbol）、**国内场外基金→天天基金/东方财富免费接口（自动净值）**、加密→CoinGecko、黄金→免费现货或手动、房产/现金外币→手动（现金外币=金额×汇率）；支持卖出操作（减数量，开联动则生成交易）；价格刷新：任意网络 + 每日定时两次(11:30、15:00，BGTask best-effort) + 手动强制；净资产 = Σ资产账户 + Σ持仓市值(基础币种折算) − Σ负债账户；可设置不计入净资产；资产↔现金流联动开关(默认关)；分红/staking 手动记(开联动自动)；**持仓强制归属某投资账户**（账户为唯一脊柱，无跨账户汇总）；资产页按账户类别分区（投资/现金/实物/负债），投资账户下嵌套持仓，点持仓进详情页看 周/月/季/年/总计 资金走向与明细（期间汇总/交易流水/占比/已实现未实现盈亏分离）
17. **负债债权**：信用卡透支/贷款/借入为负债，借出为应收；信用卡账单日+还款日，看待还金额、还款日前本地通知
18. **AI 分析（单独页）**：设置填 OpenAI 兼容接口（Key 存 Keychain + Base URL + 模型）；页内①月度文字洞察报告（手动触发 + 每月 1 号自动生成，结果缓存 7 天）②「追问」对话式问答（历史保存、可手动清除）③醒目免责声明
19. **App 锁**：Face ID/Touch ID，设置可配触发（每次冷启动 / 后台 N 分钟 / 关闭），失败转密码
20. **隐藏金额**：设置「隐藏金额」开关，一键将所有余额/金额变圆点防旁窥
21. **币种设置**：选基础币种（默认 CNY，可切 USD/HKD/JPY/EUR…）
22. **同步**：iCloud 自动同步；列表顶部显示同步状态（同步中/已同步）；App 角标显示「待确认笔数 + 即将续费数」
23. **种子数据**：首次启动幂等种子（预设大类/子类/账户/标签只种一次）
24. **上架物料**：App 名占位「记账本」；占位隐私政策文案 + 占位 URL；SF Symbol 占位图标；首次启动轻量引导页（可跳过）；启动屏

## 4. 数据模型（SwiftData / CloudKit 兼容）
- **Transaction**：kind(支出/收入/转账/退款)、amount、category、account、toAccount、date、note、tags[](→Tag)、attachments[](CloudKit Asset)、createdAt、needsReview、source、affectsAsset(默认 true)、fromRecurring、refundOf(关联原交易)
- **Category**：名称、图标(SF Symbol)、父级ID(可选, 子类)、默认类型、关键词[]、sortHint
- **Tag**：名称、颜色
- **Account**：名称、角色(.asset/.liability)、子类型、初始余额、statementDay/dueDay、archived(默认 false)、currentValueSource(现金/负债=由期初+交易推导；实物=手动填估值)
- **Budget**：月份、分类(可选)、限额、copiedFromLast
- **RecurringRule**：模板字段、频率、下次生成日、lastGenerated、dedupHash、active
- **Subscription**：名称、金额、分类、账户、模式、频率、下次续费日、lastGenerated、状态
- **Holding**：类型、代码、名称、数量、成本价(手动)、当前价、价格时间、关联账户(必填，账户归属)、quoteSource、costBasisRule(默认 weightedAvg，可切 fifo / 手动指定批次)、realizedPnl(卖出产生)
- **HoldingSnapshot**：holdingID、日期、数量、市值、成本值、未实现盈亏（每日随行情刷新捕获，**全量保留不自动聚合丢弃**；走势展示时按分段自适应降采样；安装前无数据）
- **NetWorthSnapshot**：日期、净值（每日自动记录，供曲线）
- **AIConversation**：标题、消息[]、createdAt（历史保存）
- **AppSettings**：baseCurrency、includeLiabilityInNetWorth(默认 true)、linkAssetToCash(默认 false)、appLockEnabled、appLockMode、hideAmount、quoteRefreshTimes([11:30,15:00])、language(zh/en)、AI/Mail/Market 凭据(Keychain)、通知提前天数

## 5. 界面结构（Tab）
- **记账**（首屏）：快速录入 + 本月列表 + 转账 + 待确认队列 + 全局搜索
- **统计**：月度 / 趋势 / 年度 / 占比 / 储蓄率 / 净资产曲线 / 自定义区间（可下钻）
- **资产**：净资产、账户(资产/负债，可归档)、投资持仓、行情刷新、卖出操作、负债与应收
- **订阅**：列表（三种模式）、即将续费
- **我的**：账户/分类/标签管理、预算、模板、邮箱/行情/AI/币种/App锁/隐藏金额/语言 设置、CSV/JSON 备份、AI 分析入口、清空数据(危险操作)

## 6. 自动生成引擎（统一）
- 单一 `AutoGenerator`：启动/每日前台扫描 RecurringRule 与 Subscription(周期自动)，按 dedupHash 去重生成 Transaction 并标记 lastGenerated；实例独立、改不影响模板；循环/订阅靠 active 标志手动取消。
- 订阅「手动提醒」模式：续费日前 N 天发本地通知，不生成交易。
- 每日记录一条 NetWorthSnapshot 供曲线。

## 7. 技术默认（本人替你定，可改）
- 重复防护：自动生成按「来源+商户+金额+日期」hash 去重
- 分类图标：SF Symbols；最近/常用智能置顶
- 减仓/清仓：支持部分卖出，盈亏按手动成本 × 剩余数量
- AI 报告：手动 + 每月 1 号自动生成，缓存 7 天避免重复花 Key
- 邮箱拉取：手动触发 + 前台；支持绑定多个邮箱
- 行情定时：11:30、15:00 经 BGAppRefreshTask best-effort（真机调），任意网络 + 手动
- 基金行情：天天基金/东方财富免费接口（国内场外），美股/ETF/港股/A股走 Yahoo
- 通知：订阅默认提前 1 天 + 当天；预算超支实时 + 每日汇总；提前天数可配；权限被拒应用内角标兜底
- App 锁失败：连续失败转密码
- 财政月：默认自然月，暂不做自定义周期
- 汇率源：免费汇率 API 缓存，用于 A股/HK股/现金外币折算；加密 CoinGecko 直接给本币价
- 删除：软删除（isDeleted 标记）保障 CloudKit 安全；多设备冲突 last-write-wins
- 单账本；金额输入数字键盘 + 运算；附件每笔上限 5 张并压缩
- 净资产曲线：每日自动快照一条；统计图表可分享图片
- 对账校验：账户「期初余额 + 交易净额」与当前显示值不一致时，账户行标黄提示，避免数据失真
- 卖出成本规则：默认加权平均（每笔加仓更新均价），设置可切 FIFO 先进先出 / 手动指定批次；已实现盈亏=卖出收入−对应成本；未实现盈亏=当前市值−剩余成本
- 引导页首次启动可跳过；启动屏占位
- Widget / Siri 快捷指令：未来版本，不在 MVP

## 8. 明确不做
- 逐笔多币种（统一基础币种，可切）
- 报表 PDF 导出
- 网页端/后台
- 短信/通知抓取（系统隐私限制）
- 家庭成员 CloudKit 共享
- 后台自动邮箱拉取
- 基金行情非自动（已改为自动）

## 9. 交付与运行
Windows 无法跑 Xcode。交付**完整、模块化、可编译的 Xcode 工程源码**（含 i18n、占位图标/隐私政策/引导页/启动屏）。Mac 上：Xcode 打开 → 配 Bundle ID 与开发者团队 → 启用 iCloud(CloudKit) → 编译运行/上架。Yahoo、天天基金/东方财富、CoinGecko、汇率、AI、IMAP/OAuth 六处网络模块需真机联调。
